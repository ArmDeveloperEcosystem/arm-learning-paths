---
title: "Use case: macOS smoke test (export → run → validate)"
---

This is the fastest way to prove your environment + runner builds + pipeline wiring are correct.

## Preconditions

- You are in the profiling kit root (after copying `executorch_sme2_kit/` to your workspace):
  - `.../executorch_sme2_kit/`
- Network access is available (for cloning/installing ExecuTorch).

## Run

```bash
# 1) Setup ExecuTorch + venv
bash model_profiling/scripts/setup_repo.sh

# 2) Build SME2-on/off runners
bash model_profiling/scripts/build_runners.sh

# 3) End-to-end smoke test
python model_profiling/scripts/run_quick_test.py
```

## Expected outputs

- `out_toy_cnn/artifacts/toy_cnn_xnnpack_fp16.pte`
- `out_toy_cnn/artifacts/toy_cnn_xnnpack_fp16.pte.etrecord` (optional)
- `executorch/cmake-out/mac-arm64/executor_runner`
- `executorch/cmake-out/mac-arm64-sme2-off/executor_runner`
- `out_toy_cnn/runs/mac/manifest.json` (optional)
- `out_toy_cnn/runs/mac/metrics.json` (optional)
- `out_toy_cnn/runs/mac/*/*.etdump`
- `out_toy_cnn/runs/mac/*/*.csv` (automatically generated by pipeline)
- `out_toy_cnn/runs/mac/*_pipeline_summary.json`
- `out_toy_cnn/runs/mac/analysis_summary.json` (automatically generated by pipeline)

Optional (re-run analysis if needed):

```bash
python model_profiling/scripts/analyze_results.py --run-dir out_toy_cnn/runs/mac
```

Expected output:

- `out_toy_cnn/runs/mac/analysis_summary.json` (if not already generated)
